◆ computeScores()

void OthelloBoard::computeScores ( )

This is a function to display the board on the command line.

This function clears everything previously present in the terminal, and displays new board configuration. This function calls displayScores() and displayMoves() to display scores of both the players and legal moves of the current player.

Parameters
colis used to specify whose turn it is currently.

This function calculates the current scores of the players
