Detailed Description

This class is the controller of the application. It integrates model and the view of the application. It also takes input from the user and passes information to the model.

◆ AIPlay()

void Controller::AIPlay ( OthelloBoard  board)

This function lets the AI play after every move the player makes.

If the AI is NoAI, the program waits until the second player has made a move. Otherwise, it makes a move as soon as the first player makes a move,

Parameters
boardThis is the model of the game. It has data about the present board configuration, the valid moves that AI can make, so that the AI can select the required move and place the move.
