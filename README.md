
# Othello_Game

A basic version of Terminal based Othello game

## Dependencies
- g++-10
```sh
sudo apt-get update
sudo apt install g++-10
```
- fmt library
```sh
sudo apt update
sudo apt install libfmt-dev
```

## Build
```sh
git clone https://github.com/GauthamBellamkonda/Othello_Game.git
cd Othello_Game
mkdir objs
make run      #to build and run the executable
```
